md-fire — нативный macOS-редактор Markdown
==========================================

Чтобы .md-файлы наконец-то было приятно ЧИТАТЬ, а не разглядывать звёздочки
и решётки в сером редакторе кода. Живой WYSIWYG как в Typora плюс фокус и
типографика как в iA Writer.


КАК УСТАНОВИТЬ
-------------

1.  Перетащите значок  md-fire  →  в папку  «Программы» (Applications),
    она прямо здесь, в этом окне.

2.  ПЕРВЫЙ ЗАПУСК. Приложение пока без подписи Apple, поэтому macOS
    осторожничает. Откройте его так:

        Программы → правый клик по «md-fire» → «Открыть» → ещё раз «Открыть».

    Дальше запускается обычным двойным кликом.

    Если система всё равно не пускает:
        Системные настройки → Конфиденциальность и безопасность →
        прокрутите вниз → «Открыть всё равно».


ЗАЧЕМ ОНО
---------

Проект вдохновлён болью вайбкодинга: ИИ генерит горы Markdown — планы, спеки,
README, заметки — а читать их в редакторе кода невозможно. md-fire даёт для
этого отдельное, красивое и тихое место.


ЧТО УМЕЕТ
---------

  -  Живой WYSIWYG — маркеры прячутся, текст свёрстан; файл на диске не меняется
  -  Режим честного исходника (iA-style) — маркеры видны, но приглушены
  -  Focus Mode — затемняет всё, кроме активного предложения, строки или абзаца
  -  Typewriter — строка с курсором по центру экрана
  -  Боковая панель: дерево файлов и Outline документа
  -  Bionic reading и подсветка частей речи
  -  Reading time, затухающий статус-бар, поддержка Reduce Motion
  -  Экспорт в PDF и HTML
  -  Copy for Telegram — весь документ или выделенный фрагмент

Требуется macOS 14+ (Apple Silicon).
